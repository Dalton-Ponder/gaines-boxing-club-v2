# Design System

## 6. Design System Notes for Stitch Generation

The design MUST be strictly DARK MODE ONLY. The aesthetic is 'underground style' with concrete-like textures, gritty but premium and professional. The primary accent color is Kiln Orange (#C04E01). Use highly polished UI elements, use deep dark grays/blacks (#111, #1A1A1A) with striking Kiln Orange (#C04E01) typography, buttons, borders, or glowing accents to evoke the feel of a gritty yet premium underground boxing gym. Font: LEXEND. Roundness: ROUND_EIGHT.

## Accepted Hero Styles
1. Home Page Hero: Unique, allowed to be different.
2. Legacy Page Hero: Full-screen height (h-[70vh]) with vertical gradient overlay, centered text, no CTA.
3. Philosophy Page Hero: Full-screen height (h-[70vh]) with standard background, centered text, CTA button included.
4. Schedule Page Hero: Condensed upper section with a boxed lower section containing unique information/CTA.
All other pages must use one of these accepted hero designs.
